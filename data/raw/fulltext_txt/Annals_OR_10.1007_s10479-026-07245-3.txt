Abstract
In this paper, we propose a new Shapley-type value (the 2-Shapley value) for bicooperative
games. Unlike classical cooperative games, bicooperative games assign values to pairs of
disjoint coalitions (bicoalitions) representing players who support change and those who
defend the status quo, while the remaining players abstain. To develop this new value, we
employ antimatroid theory not as a structural constraint on the games, but as a methodological
bridge. SpeciямБcally, we demonstrate that any bicooperative coalition structure is isomorphic
to a speciямБc antimatroid, which we term the тАШdouble antimatroidтАЩ. Leveraging this mathemat-
ical connection, we propose a new Shapley value, provide an axiomatization that preserves
its fundamental axioms, and establish its consistency with existing solution concepts. Finally,
we illustrate our theoretical proposal and the calculation of this value through an example
modeling a decision-making process. This work provides a robust mathematical foundation
connecting both structures, opening new perspectives for analysis within bicooperative game
theory.

1 Introduction
Game theory has emerged in recent years as a fundamental tool for economic analysis and,
in general, for decision-making. In particular, cooperative games allow for the study of
collaborative situations among a group of agents. Classical cooperative games are deямБned by
the concepts of coalition and characteristic function. A coalition is the set of agents formed
by the decision to agree or disagree on a particular joint action. The characteristic function
assigns a payoff to each possible coalition as a result of its formation. The payoffs of the
characteristic function can be understood in many quantitative ways (costs, beneямБts, debts,
etc.) but can also have a qualitative meaning, as in the case of simple games (those in which the
characteristic function has only two possible values: 0 indicating that the coalition is losing
and 1 meaning that it is winning). Based on this information, a solution to the game consists
of ямБnding a vector with the ямБnal individual payoffs each agent will receive for each game.
A value is a function that provides a speciямБc payoff vector for each game. There is a wide
range of values for cooperative games in the literature. Of all of them, the most well-known
and applied is the Shapley value (Shapley, 1953), which will be the focus of this article. The
Shapley value restricted to simple games is called the Shapley-Shubik index.
Later, the notion of cooperative games was extended to situations of partial cooperation by
restricting the set of coalitions for which the characteristic function is known. The properties
of the structure of feasible coalitions for the characteristic function led to the use of math-
ematical structures such as graphs (Myerson, 1977), precedence structures Faigle & Kern,
(1992), permission structures (van den Brink, 1997; Gilles et al., 1992), convex geometries
(Bilbao, 1998), etc. Antimatroids are dual structures of convex geometries that generalize
permission structures, as highlighted in Algaba et al. (2004). Two Shapley values have been
studied in the literature on games over antimatroids: one arises from their construction as
dual structures of convex geometries (Bilbao, 1998; Jim├йnez-Losada, 1998), and the other
from their interpretation as a generalized permission structure (Algaba et al., 2003).
Felsenthal and Machover (Felsenthal and Machover, 1997) highlighted in 1997 that in
many voting situations, the choice of agents when facing a decision is not just between
agreeing or disagreeing, but rather there are three options: being in favor, being against,
or abstaining. This idea was generalized by Bilbao (2000) in 2000 with the concept of
bicooperativegames.BicooperativegamesaredeямБnedbyacharacteristicfunctionthatassigns
value to bicoalitions. A bicoalition is a disjoint pair of coalitions representing the players in
favor of cooperation (positive action) and those against it (negative action). However, these
two sets do not necessarily form a partition of the set of players; agents external to both groups
areconsidereduninvolved,neitherfornoragainst.Thesegamescanbeusedtostudysituations
where there is a status quo in player relationships that is intended to be changed to a new one,
improving the groupтАЩs beneямБt or cost. The negative option in this case represents opposition
to changing the status quo, while the positive option supports the change. Labreuche and
Grabisch (2008) describe various applications of bicooperative games, including one for
resolving cost-sharing problems in irrigation networks. Giardino (2021) used bicooperative
games to analyze decision-making in company mergers or alliances. These applications are
added to the initial one provided in Felsenthal and Machover (1997), to adjust the power of
each group in a representative chamber for a given vote.

Several values for bicooperative games exist in the literature, attempting to extend the well-
known Shapley value for classical cooperative games Bilbao et al., (2008a); Felsenthal and
Machover (1997); Labreuche and Grabisch (2006, 2008). Felsenthal and Machover (1997)
provided a power index for ternary simple games, which can be viewed as a Shapley-Shubik
index. Labreuche and Grabisch (2006) deямБned an initial Shapley value for bicooperative
games, considering only bicoalitions that form a partition of the set of players. Later, Bilbao
et al., (2008a) introduced a Shapley-type value that uses all the information contained in
the characteristic function. Again, Labreuche and Grabisch (2008) also considered a value
applicable to each predetermined possible bicoalition, which coincides with the classical
Shapley value of a speciямБc cooperative game derived from the bicooperative one. In this
article, we propose a new Shapley value for bicooperative games, which is related to both
the Shapley values introduced in Bilbao et al., (2008a) and the one given in Labreuche and
Grabisch (2008). To establish this connection, we will use games on antimatroids as a linking
framework.
In the next section, we recall the necessary concepts and results on classical cooperative
games, games on antimatroids, and bicooperative games. In Section 3, we link the family of
bicoalitions to a speciямБc antimatroid. The application of the value for games on antimatroids
in Algaba et al. (2003), seen the antimatroid as a permission structure, deямБnes a new Shapley
value for bicooperative games in Section 4. In Section 5, a characterization of this new
value is obtained. Finally, we will discuss the relationships between this new value and other
Shapley values for bicooperative games in Section 6. We will show that the application of the
other Shapley value for games on antimatroids or convex geometries (Bilbao, 1998; Jim├йnez-
Losada, 1998) coincides with the value proposed by Bilbao et al., (2008a) for bicooperative
games.

7 Conclusions
This article introduced a new Shapley value for bicooperative games: the 2-Shapley value
(DeямБnition 5). We provided a formulation (Theorem 1) in terms of bicooperative marginal
contributions and proved that it satisямБes properties similar to the classic Shapley value (Theo-
rem 2). Additionally, the new value ямБts well within the families of probabilistic values (Bilbao

et al., 2008b) and bisemivalues (Dom├йnech et al., 2018) given in this context, as shown in
Theorem 3. We also proposed an axiomatization in Theorem 4, introducing a pair of axioms,
(BS4*) and (BS5), novel in the context of bicooperative games. The model used to deямБne
the value, by considering games over antimatroids, has allowed us to analyze the proposed
value alongside the value given by Bilbao et al., (2008a) (Theorem 5). The 2-Shapley value,
as highlighted by (BS5), provides a different treatment of abstention as opposed to voting in
favor or against. Labreuche and Grabisch (2006) proposed a solution for bicooperative games
that coincides with the application of HsiaoтАЩs value (Hsiao & Raghavan, 1993) for multi-
choice games, assigning the same weight to abstention as to voting against. The 2-Shapley
value would be related to HsiaoтАЩs value, but assigns different weights to those two options
of the players.
